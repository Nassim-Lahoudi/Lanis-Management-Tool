# Lanis-Verwaltungs-Programm
Lanis Verwaltungs Programm ist eine Desktop-Anwendung zur Verwaltung von Schulportal-Hessen-Daten über die LanisAPI. Mit sicherer Anmeldung und grafischer Oberfläche bietet sie Zugriff auf Aufgaben, Vertretungspläne und weitere Funktionen zur digitalen Organisation des Schulalltags.
